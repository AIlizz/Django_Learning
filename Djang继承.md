# 继承的定义：
模板继承和类的继承含义是一样的，主要是为了提高代码重用，减轻开发人员的工作量。<br>
典型应用：网站的头部、尾部信息。<br>
可以参考如下链接：https://www.jianshu.com/p/7934914c87ee

# 父模板
```html
Header
{# 以上被继承 #}
{%block 名称%}
预留区域，可以编写默认内容，也可以没有默认内容
{%endblock  名称%}
{# 以下被继承 #}
Ender  
```

# 子模板

```html
{% extends "父模板路径"%} {# 必须在第一行 #}
{%block 名称%}
实际填充内容
{{block.super}}用于获取父模板中block的内容
{%endblock 名称%}

```
